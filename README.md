A missing diary taking website.

# Requirements

node v6.9.1 or above, npm, mongod and supervisor installed.

# Getting started

`npm install`

`mkdir db && mongod --dbpath db`

`supervisor --harmony index`

Now, you can see the website running on http://localhost:3000
